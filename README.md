## CLI

| Command  | Description                                |
| -------- | ------------------------------------------ |
| `start`  | Start a local development server.          |
| `build`  | Run a production build.                    |
| `test`   | Run tests.                                 |
| `test:w` | Run tests continously.                     |
| `lint`   | Lint the code.                             |
| `format` | Format the code to the project code style. |
